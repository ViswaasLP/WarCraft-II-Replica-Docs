std::string CConnection::SerializeCommand ( SPlayerCommandRequest  command,
int  playernumber 
)
static

Serialize objects of the type SPlayerCommandRequest.

The format of the serialization is as follows: All values will be converted to integer types. The '|' character will separate fields while the ',' character will separate values in list format. PlayerNumber|DPacketType|DAction|CPlayerAsset1,CPlayerAsset2...|DTargetNumber|DTargetType|DTargetLocationX,DTargetLocationY

Parameters
commandA command from the DPlayerCommands array.
Returns
A string version of the data contained in command.
