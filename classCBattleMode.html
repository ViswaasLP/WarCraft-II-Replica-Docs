void CBattleMode::Calculate ( std::shared_ptr< CApplicationData >  context)
overridevirtual

This function handles the next state of the game.

context holds the current state of the game. Based on the current state this function determines if any changes need to be made to the state. If any changes needed to be made then the context is changed.
